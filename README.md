# 🚀 Artiva – AI Image Discovery Platform

Artiva is an AI-powered image browsing web app built with Next.js.  
Users can explore AI-generated images, view details, and manage their profile with a clean and responsive UI.

---

## 🔗 Live Demo

https://artiva-ai.vercel.app/

---

## 📂 Repository

https://github.com/Tahaimage8/Artiva

---

## ✨ Features

- Next.js App Router  
- User authentication (Sign In / Sign Up)  
- Browse all images  
- Single image details page  
- Featured images section  
- User profile update  
- Responsive design  
- Uses JSON data  

---
